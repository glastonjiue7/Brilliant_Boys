# BBC Cricket Tournament 2026 - Registration Website

A professional static HTML website for cricket tournament registration, featuring all tournament rules, team registration forms, and contact information.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Cricket Tournament Rules** - All rules in Marathi (स्पर्धेचे नियम)
- **Interactive Registration Form** - Dynamic team and player registration
- **Photo Gallery** - Tournament images and gallery
- **Professional Styling** - Modern gradient design with smooth interactions
- **Fast & Lightweight** - Pure HTML/CSS/JavaScript, no dependencies

## Contents

- `index.html` - Main website with registration form and rules
- `images/` - Tournament photos and media files
- `vercel.json` - Vercel deployment configuration

## How to Deploy on Vercel

### Option 1: Using GitHub (Recommended)

1. Create a GitHub repository
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Cricket tournament website"
   git remote add origin https://github.com/YOUR_USERNAME/cricket-tournament-website.git
   git push -u origin main
   ```

2. Go to [vercel.com](https://vercel.com)
3. Sign in with GitHub
4. Click "New Project"
5. Select your repository
6. Click "Deploy"

### Option 2: Using Vercel CLI

1. Install Vercel CLI
   ```bash
   npm install -g vercel
   ```

2. Deploy from the project directory
   ```bash
   cd cricket-tournament-website
   vercel
   ```

3. Follow the prompts to complete deployment

### Option 3: Direct Upload

1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Select "Other"
4. Upload the entire `cricket-tournament-website` folder
5. Click "Deploy"
